# Dev Notes

- Backend (Express) in backend_fastapi
  - npm install
  - create .env from .env.example and set SUPABASE_URL, SUPABASE_KEY
  - npm run dev

- Frontend (React) in music-streamer-platform-297904-297913/frontend_react_js
  - npm install
  - npm start
  - Uses backend at http://localhost:3001/api by default. Configure with REACT_APP_API_BASE.
